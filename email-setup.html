You need to create a special App Password for the Lodge Finder server to send emails on your behalf.

üîó Click here to generate App Password
  • Make sure 2-Factor Authentication is enabled on your Google account

2

Edit the .env File

The .env file should already be open in Notepad. If not, open it manually.

Find this line:

EMAIL_PASSWORD=your-app-password-here

Replace your-app-password-here with your App Password:

EMAIL_PASSWORD=abcdefghijklmnop
üìù Note: Remove all spaces from the App Password when pasting it!

Save the file (Ctrl+S) and close Notepad.

3

Restart the Server

The server needs to be restarted to load the new email configuration.

  1. Go to the terminal where npm start is running
  2. Press Ctrl + C to stop the server
  3. Run npm start again
